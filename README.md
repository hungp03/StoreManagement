Ứng dụng quản lý cửa hàng - Đồ án lập trình hướng đối tượng
============================

## Sử dụng
Java, Netbeans IDE, MS SQL Server 

## Chức năng 
    # Admin
    - Toàn quyền sử dụng ứng dụng (quản lý nhân viên, nhà cung cấp, khách hàng, sản phẩm, category, invoice...)
    
    #Nhân viên kho
    - Trang chủ
    - Quản lí product
    - Quản lí category
    
     #Nhân viên bán hàng
    - Có hầu hết các tính năng của Admin, ngoại trừ thao tác với nhân viên và nhà cung cấp
    - Chỉ được xem category, product, không được can thiệp sâu
    - Bán hàng
    - Quản lí hóa đơn, xuất thông tin hóa đơn ra pdf, xlsx...
  

## Điểm trừ

Nên thiết kế thêm bảng History để lưu trữ lịch sử thao tác, hoạt động ... của app. Thêm chức năng chấm công cho nhân viên, chức năng hoàn trả sản phẩm/đơn hàng

Còn một số lỗi vặt, code chưa được tối ưu
